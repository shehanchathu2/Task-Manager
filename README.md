**Project Overview
The Task Manager Application is a full-stack web application that allows users to manage their tasks efficiently. Users can create, update, delete, and track tasks with different priorities and statuses.
The application supports user authentication using JWT, ensuring secure access to task data. Tasks can be filtered by priority and status, and the system supports role-based access control, allowing administrators to view all tasks.

Key features
    •	User Registration and Login
    •	JWT-based Authentication
    •	Create, Update, Delete Tasks
    •	View tasks with pagination
    •	Filter tasks by priority and status
    •	Role-based access (Admin/User)
    •	Responsive frontend dashboard
    •	RESTful API architecture
